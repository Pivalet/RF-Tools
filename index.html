Antenna BW & FOV

Calculates the 3dB Beam Width (BW) and the Field of View (FOV).
BW ≈ 50.7 · λ / (N · d)
FOV ≈ 50.7 · λ / d

Phase & Time Info

Relationship between phase shift and time delay at a specific frequency.
Δφ (deg) = 360 · f · t
t (sec) = Δφ / (360 · f)

Phase Correction

Matches multiple phase measurements to solve for the time-domain shift.
Δφ ≈ (360 · f · Δt) mod 360

Power Formulae

dBm to Watt: P(W) = 10(PdBm-30)/10

Watt to dBm: P(dBm) = 10 · log10(PW · 1000)

Free Space Path Loss (FSPL)

FSPL estimates signal attenuation between two antennas in a vacuum/ideal line-of-sight.
FSPL (dB) = 20log10(d) + 20log10(f) + 20log10(4π/c)

Wavelength (λ)

Distance between repeating peaks of the wave.
λ = c / f

Doppler Shift Explanation

The Doppler effect is the change in frequency observed when a source and an observer move relative to each other.

If the source moves towards the observer, the waves are "compressed," resulting in a higher perceived frequency. If moving away, waves are "stretched," resulting in a lower frequency. This shift is critical in radar velocity measurements.
Δf = (v / c) * f0

Phased Array Parameters

1. Antenna Gain (G): Represents the array's ability to focus energy in a specific direction.
G = 10 · log10((4π · A · η) / λ²)
(A = Area, η = Efficiency)

2. Half-Power Beamwidth (HPBW): The angular width where the power is half the peak (-3 dB).
HPBW ≈ 50.7 · (λ / L)
(L = Total length of array in that dimension)

3. Antenna Pattern (AF): The Array Factor determines the interference pattern of N elements.
AF = |sin(Nψ/2) / (N · sin(ψ/2))|
(ψ = phase shift based on spacing and scan angle)

Far Field (Fraunhofer Distance)

The far-field region is where electromagnetic waves become nearly planar and antenna radiation characteristics no longer depend on distance.
R = 2D² / λ
D = antenna diameter
λ = wavelength
λ = c / f
